# Деплой на Beget (VPS / Linux)

Приложение изначально настраивалось под Windows (`C:\WND`). На сервере Beget (Linux) нужны отдельные шаги.

## Почему возникают проблемы

| Симптом | Причина |
|--------|---------|
| «Обзор» в настройках не работает | Диалог папки — только Windows с рабочим столом. На Linux путь вводится вручную |
| ГОСТ не индексируется | Пустая папка `FZYur` или в `config.cfg` указан путь `C:\WND` |
| PDF в квадратиках | Не подключён шрифт с кириллицей / эмодзи на сервере |
| PDF не сохраняется в OUT | Неверный `work_folder` или нет прав на запись |

## 1. Загрузка проекта на сервер

Структура на сервере (пример):

```
/home/ваш_логин/wnd/
├── backend/
├── frontend/
├── config.cfg
├── .env
├── FZ/
├── FZYur/      ← сюда положить файлы ГОСТ
├── IN/
├── OUT/
└── new-doc/
```

## 2. Переменные окружения (`.env`)

```env
OPENAI_API_KEY=sk-...
SECRET_KEY=случайная_строка
# Опционально — рабочая папка (если не через config.cfg):
WND_WORK_FOLDER=/home/ваш_логин/wnd
```

## 3. Настройка `config.cfg`

На Linux **не используйте** `C:\WND`. Укажите путь на сервере:

```ini
[paths]
work_folder = /home/ваш_логин/wnd
```

Сохраните через ⚙ Настройки или отредактируйте файл и перезапустите backend.

## 3.1. Мультиплатформенная загрузка файлов с ПК

На Beget кнопка системного выбора **рабочей папки сервера** недоступна, но загрузка файлов с вашего ПК работает через браузер:

1. Откройте ⚙ **Настройки**.
2. В поле **Рабочая папка** вручную укажите серверный путь, например `/home/ваш_логин/wnd`.
3. В блоке **Загрузка файлов с вашего ПК** выберите назначение:
   - `FZYur` — для ГОСТ и стандартов
   - `FZ` — для федеральных законов
   - `IN` — для ВНД
   - `OUT` — для отчётов/результатов
   - `new-doc` — для проектов новых ВНД
4. Нажмите **Выбрать файлы…** или **Выбрать папку…**.

Выбор папки работает в Chrome/Edge/Chromium через браузерный `webkitdirectory`. Реальный путь на вашем ПК браузер не передаёт — на сервер уходят сами файлы и относительная структура папки.

## 4. Установка зависимостей

```bash
cd /home/ваш_логин/wnd/backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Рекомендуется установить шрифты (для PDF):

```bash
sudo apt-get update
sudo apt-get install -y fonts-dejavu fonts-noto-color-emoji
```

## 5. Папка ГОСТ (FZYur)

1. Загрузите на сервер файлы ГОСТ (`.pdf`, `.docx`, `.txt`) в папку `FZYur/` через ⚙ Настройки → **Загрузка файлов с вашего ПК** или через SFTP.
2. На главной странице в блоке **Статистика** нажмите пересоздание базы **ГОСТ**  
   или выполните:

```bash
curl -X POST http://127.0.0.1:8011/api/bases/gost/recreate
```

3. Проверьте статус:

```bash
curl http://127.0.0.1:8011/api/bases/status
```

В ответе у `gost` должны быть `ready: true` и `files_count > 0`.

## 6. Запуск backend

```bash
cd /home/ваш_логин/wnd/backend
source .venv/bin/activate
uvicorn main:app --host 0.0.0.0 --port 8011
```

Для постоянной работы — systemd, supervisor или панель Beget.

Прокси nginx/apache на Beget должен направлять `http://185.23.34.238/` → `http://127.0.0.1:8011`.

## 7. Права на запись

```bash
chmod -R u+rwX /home/ваш_логин/wnd/OUT
chmod -R u+rwX /home/ваш_логин/wnd/backend/chroma_db
chmod -R u+rwX /home/ваш_логин/wnd/IN
```

## 8. PDF-отчёты

- **Скачать** — файл уходит в браузер (папка Downloads на вашем ПК).
- **Сохранить** — файл пишется в `OUT/` на **сервере** (не на ваш компьютер).

После обновления кода на сервере PDF должен отображать кириллицу (DejaVu/Noto).  
Цветные эмодзи на Linux могут заменяться текстом `[OK]`, `[X]` — это нормально.

## 9. Обновление кода на Beget

Скопируйте на сервер обновлённые файлы:

- `backend/path_config.py`
- `backend/report_pdf.py`
- `backend/main.py`
- `frontend/config-ui.js`
- `frontend/index.html`

Перезапустите uvicorn и обновите страницу в браузере (**Ctrl+F5**).

## 10. Быстрая проверка

```bash
curl http://127.0.0.1:8011/api/config
curl http://127.0.0.1:8011/api/bases/status
```

В `config` проверьте `work_folder` (Linux-путь) и `browse_folder_available: false`.  
В `bases/status` — готовность `gost`, `fz`, `vnd`.
