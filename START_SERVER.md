# Инструкция по запуску сервера

## Способ 1: Через run.py (рекомендуется)

```bash
cd D:\WND
python run.py
```

## Способ 2: Напрямую через main.py

```bash
cd D:\WND\backend
python main.py
```

## Способ 3: Через uvicorn напрямую

```bash
cd D:\WND\backend
uvicorn main:app --host 0.0.0.0 --port 8011 --reload
```

## Проверка работы сервера

После запуска сервер должен быть доступен по адресу:
- http://localhost:8011
- http://127.0.0.1:8011

## Возможные проблемы

### Порт 8011 занят
Если порт 8011 уже занят другим приложением:
1. Закройте другое приложение, использующее порт 8011
2. Или измените порт в `run.py` или `main.py`

### Ошибка подключения к базе данных
Убедитесь, что в файле `.env` указан правильный `DATABASE_URL`:
```
DATABASE_URL=sqlite:///D:/WND/backend/wnd.db
```

### Модули не найдены
Убедитесь, что вы активировали виртуальное окружение:
```bash
cd D:\WND\backend\.venv\Scripts
activate
```

## Логи

При запуске сервера вы увидите:
- Информацию о инициализации базы данных
- Информацию о векторных базах
- Сообщения о готовности сервера

Если возникают ошибки, они будут выведены в консоль.
